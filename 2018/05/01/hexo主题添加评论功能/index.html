[post]hexo主题添加评论功能

2018-05-01  

前言：本文简要介绍livere评论系统的的添加，如您想要添加其他评论系统，请拉至文末，阅读相关博文（或者查阅更多资料）

一、注册评论系统

动手前请阅读网友博客：传送门

二、修改相关文件

1、修改配置文件 D:\hexo\themes\indigoCard\_config.yml，添加如下代码：

#来必力评论系统
livere_uid: true

2、修改脚本文件 D:\hexo\themes\indigoCard\layout_partial\post\comment.ejs，添加如下代码：

<%- partial('../plugins/livere') %>

3、添加脚本文件 D:\hexo\themes\indigoCard\layout_partial\plugins\livere.ejs，代码如下：

<!-- livere BEGIN -->

<div id="lv-container" data-id="city" data-uid="MTAyMC8zNjE0Mi8xMjY3Nw==">
    <script type="text/javascript">
   (function(d, s) {
       var j, e = d.getElementsByTagName(s)[0];

       if (typeof LivereTower === 'function') { return; }

       j = d.createElement(s);
       j.src = 'https://cdn-city.livere.com/js/embed.dist.js';
       j.async = true;

       e.parentNode.insertBefore(j, e);
   })(document, 'script');
    </script>
<noscript> 为正常使用来必力评论功能请激活JavaScript</noscript>
</div>

<!-- livere END -->

简单而言，其中代码来自于官方网站。

三、重新部署

hexo generate
hexo deploy
