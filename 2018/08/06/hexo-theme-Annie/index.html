  • js静态切换图片（刷新一次换一张）
<script type="text/javascript">
    //数组的长度=图像的数量
    var bodyBgs = [];
    bodyBgs[0] = "img/01.jpg";
    bodyBgs[1] = "img/02.jpg";
    bodyBgs[2] = "img/03.jpg";
    bodyBgs[3] = "img/04.jpg";
    bodyBgs[4] = "img/05.jpg";

    var randomBgIndex = Math.round( Math.random() * 4 );

    //输出随机的背景图
    document.write('<style>body{background:url(' + bodyBgs[randomBgIndex] + ')}</style>');


</script>

  • 提交本地代码到远程仓库（git——>github）
git remote add origin https://github.com/Sariay/hexo-theme-Annie.git
//只需关联一次即可

提交单个文件

$ cd hexo-theme-Annie
$ touch README.md
$ git init
//添加README.md、初始化仓库

$ git add README.md
$ git commit -m "first commit" //提交说明
//提交README.md文件到“本地版本库”

git remote add origin https://github.com/Sariay/hexo-theme-Annie.git
git push origin master
//关联仓库、提交本地代码到远程仓库

提交全部文件

$ cd hexo-theme-Annie
$ touch README.md
$ git init
//添加README.md、初始化仓库

$ git add .    
//提交全部文件到暂存区
$ git commit -m "xxx commit" //提交说明
//提交全部文件到“本地版本库”

git remote add origin https://github.com/Sariay/hexo-theme-Annie.git
git push origin master
//关联仓库、提交本地代码到远程仓库
